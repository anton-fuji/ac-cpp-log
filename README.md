# ac-cpp-log
## Setup Cpp

### コンパイラ比較: `gcc` / `g++`

| 特徴               | **gcc**                               | **g++**                                       |
|:------------------|:------------------------------------|:----------------------------------------------|
| デフォルト言語     | C                                   | C++                                           |
| インクルード検索   | C 用ヘッダのみ (`/usr/include` 等) | C++ 標準ヘッダ (`<vector>`/`<string>`等) を含む |
| ライブラリリンク   | 手動で `-lstdc++` が必要             | libstdc++ 等 C++ ランタイムを自動リンク        |
| ファイル拡張子判定 | `.c`/`.s` を C と判定               | `.cpp`/`.cc` を C++ と判定                    |

#### Example

```bash
# C++ ファイルをコンパイル
g++ main.cpp -o main

# C ファイルを C++ としてコンパイル
g++ foo.c -o foo

# gcc で C++ ランタイムを手動リンク
gcc main.o -lstdc++ -o main

```
